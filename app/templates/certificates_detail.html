How certificates work here

  1. Vault — this page holds one TLS identity (fullchain + key), encrypted. Never shown again.
  2. Self-managed edge map — optional mapping to this PiHerder’s Caddy (./certs); renew re-applies while enabled.
  3. Fleet service maps — each remote consumer: host + path + write mode + restart.
  4. Deploy — edge in-stack, or SSH write for fleet maps.
