Set up a managed certificate

Three steps: put PEMs in the vault, map where they land on a host, then Deploy over SSH. This is the fleet vault — not PiHerder’s own Caddy HTTPS (operator docs: Trusted HTTPS vs managed certificates).

All certificates
{# Step progress #}
  1. Step 1 · Vault
    Import certificate

    Pull from Nginx Proxy Manager or upload PEM fullchain + key (encrypted immediately).

  2. Step 2 · Map
    Add a service map

    Pick a preset (NPM, OctoPi, Grafana, UniFi…) then adjust host path and restart command.

  3. Step 3 · Deploy
    Write files to host

    Deploy writes PEMs, sets mode/owner, runs your post-deploy command. Confirm files on the host.
