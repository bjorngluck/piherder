### Conventions

- Prefer **light theme** for print clarity; dark optional as `*-dark.png`.
- Width ~**1400–1600px**; crop browser chrome if noisy.
- Redact hostnames/IPs if public screenshots go on the marketing site.
- Wireframe SVGs (`*.svg` in this folder) ship as interim placeholders until PNGs exist.
